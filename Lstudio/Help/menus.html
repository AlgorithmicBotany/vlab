L-studio menu commands

In addition, when an object has been opened, more commands are available in the Object and Tools menus.

Object

The Object menu has the following options:

  • New creates a new object. Until it is saved, the object is not connected to any object directory; it is only held on the lab table.

  • Open brings up the Browse for object dialog box. You can browse to an object's directory and select it to open the object in L-studio.

  • Save, Save as, and Make extension are described in the section Saving objects.
  • If the object has been opened with the browser, then the command Position in browser selects and shows the object's position in the browser.
  • Create a demo file is currently unsupported.
  • Write to lab table and Read from lab table write and read the current versions of the components in the L-studio editors to and from the lab table, respectively. This is useful if you use an external editor, or the L-studio generic text editor, to change an object component, and want the L-studio version to reflect these changes.
  • Show lab table opens a Windows file browser which shows the lab table of the current object.
  • Close closes the current object. If it has not been saved, you will be prompted if you want to save it.
  • Exit quits L-studio. All objects will be closed, so if you have not saved them, you will be prompted to do so.

Cpfg / Lpfg

The Cpfg/Lpfg menu is only present when an object has been opened. The menu title depends on the current object: if the current object uses cpfg, the title is Cpfg; if it uses lpfg, the title is Lpfg. The menus are otherwise identical.

  • Go! runs the simulator (cpfg or lpfg). If the simulator is already running, this command is ghosted out.
  • Kill closes the simulator. If the simulator for this object is not currently running, this command is ghosted out.
  • Continuous modeling and New... are discussed in Running a simulation.
  • Rerun reruns the simulation.

Preferences

The Preferences menu is discussed completely in the section Configuring L-studio.

Tools

The Tools menu lets you access some peripheral programs, including the object browser.

  • Snap icon is discussed in the section Making your own icons. It is available only when an object has been opened.
  • The Browser submenu contains commands which let you start the object browser. See Opening the object browser for more information.

Window

The Window menu has commands for changing the display of L-studio's object windows.

  • Tile Horizontally arranges multiple object windows by making them the full width of the L-studio window, and stacked on top of one another:

  • Tile Vertically arranges multiple object windows by making them the full height of the L-studio window, and placed in a row:

  • Cascade arranges multiple object windows by making them smaller than the L-studio window, and partially overlapping them:

  • Close all closes all of the currently opened objects. You may be prompted to save them.
  • At the bottom of the Window menu is a list of all of the currently open objects. Selecting one brings it to the front.

Help

The Help menu's commands show all of the manuals in the Help directory:

  • L-studio manual is this HTML documentation.
  • Lpfg manual is the LPFG user's manual.
  • On the Cpfg manuals submenu are:
    • User's manual, the CPFG user's manual;
    • Environmental programs, which displays the user's manual for environmental programs; and
    • Graphics extensions, which shows Extensions to the graphical interpretation of L-systems based on turtle geometry.
  • Finally, About shows some information about L-studio and cpfg's creators.
