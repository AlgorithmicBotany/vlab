Palette Quick Help

  • Select color: Left click
  • Interpolate between colors:
    • Left click and drag to select a range, and
    • select "Interpolate" from the pop-up menu
  • Copy and paste (within the Palette window):
    • Left click on a color or drag to create a range,
    • select "Copy" from the pop-up menu,
    • left click on the target color (or first color in a range), and
    • select "Paste" from the pop-up menu
