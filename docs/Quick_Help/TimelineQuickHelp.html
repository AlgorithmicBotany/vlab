Timeline Quick Help

In Execute mode

  • Move point: Left click and drag
  • Move multiple points together:
    • Shift + Left click to select each point, then
    • Left click and drag on any selected point
  • Deselect point(s): Click anywhere except on a point

  • Zoom: Cmd + Left click and drag
  • Pan: Left click and drag anywhere except on a point

  • Open function editor: Left click on name

In Edit mode

  • Move a timeline up/down:
    • Left click on name to select, then
    • use up/down arrow keys
  • Edit timeline: Left double-click on name
