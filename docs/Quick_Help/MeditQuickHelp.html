Medit Quick Help

Main window

  • Select material: Left click
  • Open editor window: Left double-click
  • Interpolate between colors:
    • Left click and drag to select a range, and
    • select "Interpolate" from the pop-up menu
  • Copy and paste (within Medit window(s)):
    • Left click on a material or drag to create a range,
    • select "Copy" from the pop-up menu,
    • left click on the target material (or first material in a range), and
    • select "Paste" from the pop-up menu

Editor window

  • Edit color component:
    • Value: Adjust slider
    • Open Color Pick window:
      • Left click on the rectangle to the left of a component
