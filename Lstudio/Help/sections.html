Editors

The object window in L-studio is divided into sections identified by tabs:

Each tab is associated with an editor:

  • Text editors:
    • L-system editor
    • View file editor
    • Description editor
    • General text file editor
  • Form-based editors:
    • Animation file editor
  • Palette editors:
    • Color map editor
    • Material editor
  • Editors with galleries:
    • Basic surface editor
    • Advanced surface editor
    • Contour editor
    • Function editor
    • Panel editor
