Quick Help for VLAB Material Editor

1 Medit Execution

Execute the VLAB Material Editor using the following arguments:

>  medit
Open an empty gallery for up to 256 materials.

>  medit [file.mat] ...
Opens the specified material file(s).
If the file does not exist, creates new file.
If a list of material files is specified, each is open in a separate application window.

>  medit ... -w [width] [height]
Opens medit with specified dimensions
 

2 Display and Selection

  • When the program starts, 16 of the 256 spheres are displayed, beginning on the first page. Index 0 is in the upper left corner, with consecutive entries vertically, top to bottom.  To display more or less spheres per page, select Display from the menu bar at the top of the window, then select the desired page size (1, 4x4, 8x8, 16x16).
  • A material index may be selected by clicking the left mouse button on the entry, or by entering the index in the first spinbox in the lower right corner.
  • A range of indices can be selected by clicking the left mouse on the lesser index, and dragging to the greater index.  A range can also be selected by selecting Edit from the menu bar, and then selecting Select Range..., which opens a dialog box to input the start index and end index or number selected.
  • All materials on the current page, or all materials in the gallery, can be selected by selecting Edit from the menu bar, then Select All in Page or Select All in Gallery.
  • Selected indices are outlined in red.
  • A page may be jumped to by entering the page number in the second spinbox in the lower right corner, or by selecting Display from the menu bar and then selecting Next Page, Previous Page or First Page.  The number of pages, as well as the page a particular material falls on, depends on the page size.  The indices materials that are displayed on the current page are shown in the upper right corner of the window.  When a page is changed, the current selection is not affected.
  • The background is black by default.  To show transparency, it is useful to change the background to a checkerboard pattern.  To do this, select Display from the menu bar, then select Show Background.  To return to a solid black background, select this option again.
  • The spheres are rendered as a polygon mesh between vertices.  The rendering speed varies inversely to the smoothness of the spheres.  To adjust the smoothness, and to set the auto-adjustment of the smoothness, select Display from the menu bar, then select Image Quality... to edit the smoothenss parameters.  Once the display is satisfactory, the parameters may be written to the configuration file by clicking the button marked Save.


2.1 Actions on Selection

The Popup Menu is activated by clicking the right mouse button anywhere on the material gallery.  The popup menu contains actions to be performed on the selection.
The same options are available by selecting Edit from the menu bar.
 

Cut, Copy, Paste, Insert

  • Cut, Copy, Paste, and Insert can be invoked by activating the popup menu or by selecting Edit from the menu bar, and then selecting the desired option.
    • If a selection is cut, all following entries are shuffled up, and the gallery is padded with default materials.
    • If a selection is inserted, all following entries (including the current selection) are shuffled down, and extra entries are pushed out.
    • If the selection that is inserted or pasted is larger than the space remaing in the gallery, the extra entries are ignored.


    Interpolation
    All material properties can be  interpolated over  a selected a range of indices by activating the popup menu or by selecting Edit from the menu bar, and then selecting Interpolate.
     

    Set to Default
    A selected index or range of selected indices can be set to the OpenGL default  material by activating the popup menu or by selecting Edit from the menu bar, and then selecting Set to Default.
    This can also be done through the M-Edit Dialog (described below) by clicking the button marked Default.
     

    M-Edit
    A material editing dialog (see below) is opened for the selected material (or a range of materials) by activating the popup menu by right clicking on the gallery, or by selecting Edit from the menu bar, and then selecting M-Edit.
     

    2.2 Material Edit Dialog

    A preview of the selected sphere is displayed on the left of the dialog.
    The material properties ambient, diffuse, specular, emissive, shininess, and transparency can be edited using the sliders or by entering a value in the edit boxes to the right of the sliders (the return key must be pressed after editing in the boxes).
     

    Material Properties

    • Ambient, diffuse,  specular and emissive are values between 0 (none) and 1 (full).
    • Shininess is a value between 0(non-shiny) and 128(max-shiny).  If a material has a specular property, for best effect modify shininess as well.
    • Transparency is a value between 0(opaque) and 1(transparent).


    Colour Swatches

    • The colour swatch popup menu is activated by pressing the right mouse button.
    • The colour of ambient, diffuse,  specular and emissive can be edited by pressing the left mouse button over the colour swatch to the left of the property labels, or by activating the popup menu and  selecting the option Edit from the popup menu.  The HSV Colour Picker is opened to edit the colour.
    • The left side of the colour swatch displays the actual colour at the current value of the material property.  The smaller section on the right displays the full intensity of the colour.
    • The full intensity of the colour can be copied and pasted to another colour swatch on the dialog.
    • If the material property has a value of 0, the full intensity is white, and the actual colour is black.


    Modifications and Action Buttons

    • If the selected material has been modified, the user cannot select another material index in the gallery until the modification has been applied or undone by clicking the button marked Apply or the one marked Undo.
    • A label displays Modified below the preview if the dialog material has been modified.
    • If the button marked Default is clicked, the material is set to the OpenGL default material.
    • If the dialog is closed by clicking the button marked Close, the material remains as it appears in the gallery.


    2.3 HSV Colour Picker

    • The HSV Colour Picker is invoked by pressing the left mouse over a colour swatch, or by activating the popup menu and  selecting the option Edit from the popup menu.
    • The colour being selected is the actual colour (rather than the full intensity).  The intensity of the material property can be adjusted by moving the slider labelled V on the left, or by moving the colour value slider on the right of the colour wheel, or by moving the slider associated with the material property on the M-Edit Dialog.
    • The colours HSV or RGB values can be adjusted using the sliders on the left side of the dialog, or by entering the value in the edit boxes to the right of the sliders (the return key must be pressed after editing in the boxes).
    • The hue and saturation of the colour can be selected on the rainbow colour wheel with the mouse by pressing the left mouse button to select a colour, and moving the mouse to adjust the selection.  The hue line and saturation circle are drawn to aid in the selection.
    • The large box below the colour wheel shows the current selection.  The selection is automatically applied.
    • The smaller box below the colour value slider shows the original colour.  To revert to this colour, left click inside this box.
    • If the colour picker is closed by clicking the button marked Close, the material property colour remains as it appears in the colour swatch on the M-Edit Dialog.

    3 File Options

    New Opens a new material gallery in the current VLAB Material Editor.
    New Window Opens a new material gallery in a newVLAB Material Editor application window.
    Load... Opens a file dialog in which the user can select an existing file to load in the current VLAB Material Editor.
    Load Window... Opens a file dialog in which the user can select an existing file to load in a newVLAB Material Editor application window.
    Save Saves the current material file.  If no file has been specified, a file dialog is opened.
    Save as... Opens a file dialog in which the user can specify a new file to save the current material file as.
    Revert to Saved Rereads the material file as it was last saved and restores the materials in the gallery.
